# Links Page — Vinicius de Morais Garcia

Página de links pessoais desenvolvida com HTML, CSS e JavaScript puro. Funciona como um hub central para acessar minhas redes sociais e projetos.

## Preview

<img width="1187" height="705" alt="image" src="https://github.com/user-attachments/assets/59a93602-1cad-4cd8-a899-2bfbe6762af6" />

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas e animações
- JavaScript puro (vanilla)
- Google Fonts — [Outfit](https://fonts.google.com/specimen/Outfit)

## Funcionalidades

- Design dark mode com card centralizado
- Animação de entrada `fadeInUp` ao carregar a página
- Navegação pelos links via teclado (setas ↑ ↓ + Enter)
- Layout responsivo, adaptado para mobile e desktop
- Hover com destaque em verde nos botões de link

##  Atalhos de teclado

| Tecla | Ação |
|-------|------|
| `↓` | Próximo link |
| `↑` | Link anterior |
| `Enter` | Abrir link selecionado |

## Paleta de cores

| Variável | Cor | Uso |
|----------|-----|-----|
| `--bg-page` | `#141414` | Fundo da página |
| `--bg-card` | `#1e1e1e` | Card principal |
| `--bg-link` | `#2a2a2a` | Botões de link |
| `--cor-destaque` | `#c5f230` | Subtítulo e hover |
| `--texto-primario` | `#ffffff` | Títulos |
| `--texto-secundario` | `#888888` | Texto auxiliar |
